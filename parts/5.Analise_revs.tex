\section{Análise comparativa entre os artigos revisados}
\label{sec:cp}

Seguindo as revisões estabelecidas nas \cref{sec_rt,sec_rssfi}, algumas possíveis frentes para pesquisa, implementação virtual e, posteriormente, implementação prática, podem ser estabelecidas. Aqui não serão discutidos ou reapresentados os artigos da \cref{sec:travel-time_tomography}, dado que os artigos lá mostrados são muito semelhantes entre si, e usados para estabelecer o modelo-padrão da \cref{subsec:sec2:modelo-padrao} e \cref{app:modelo-padrão-TTT}.

Uma delas é adotar uma abordagem diferente para o problema de inversão. Embora o TTT tradicional seja a técnica mais comum para realizar inversão utilizando somente informação de tempo de trânsito, ela não é a única capaz disso. Estas abordagens alternativas são extraídas dos artigos selecionados anteriormente nas \cref{sec_rssfi:subsec_deeper-analysis,sec_rt:subsec_deeper-analysis}. Enquanto os artigos na \cref{sec_rt} utilizam estritamente informações de tempo de trânsito (embora não necessariamente utilizando a estrutura da TTT), os da \cref{sec_rssfi} são muito mais amplos, possivelmente utilizando diferentes informações para a inversão; embora os selecionados na \cref{sec_rssfi:subsec_deeper-analysis} sejam baseados apenas em dados de tempo de percurso.

\subsection{Classificação dos artigos/técnicas}
\label{subsec:cp_classif}

Todos os artigos revisados (não somente os destacados) serão classificados, conforme as características dos métodos propostos e empregados, tipos de dados usados, entre outras categorias. Todos os artigos\footnote{Os artigos da \cref{sec_rt} estarão em azul, os da \cref{sec_rssfi} em vermelho; os artigos selecionados de cada seção estarão em negrito.} serão caracterizados pelas seguintes informações:
\begin{enumerate}[itemsep=-1ex, partopsep=0pt]
	\item Abordagem ou método (modelo direto, funções-base, parametrização, \textit{machine learning})
	\item Dimensão da aplicação (1D $\sim$ 4D)
	\item Aplicação (acústica submarina rasa, acústica submarina profunda, estruturas geológicas, biomédica)
	\item Dados de entrada (tempo de trânsito, forma de onda, modelo inicial)
	\item Parâmetros ajustáveis (vagarosidade, pesos de funções-base, parâmetros)
	\item Métrica de erro ou método de avaliação
\end{enumerate}
